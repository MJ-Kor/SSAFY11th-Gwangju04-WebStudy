# 🙌 Spring 적용 프로젝트
- **🖥PJT명**: **Enjoy Trip Project**
- **🚶‍♂️단계**: **Spring framework PJT**
- **📅진행일자**: **2024.04.25**
- **👩‍💻팀원:** **김민주, 이승원**

  
- ## 프로젝트 설명
    - Web MVC Architecture를 이해하고 활용할 수 있다.
    - Spring Framework와 RestAPI를 활용하여 웹 서버를 구축할 수 있다.
    - myBatis Framework를 활용하여 DB를 연동할 수 있다.
