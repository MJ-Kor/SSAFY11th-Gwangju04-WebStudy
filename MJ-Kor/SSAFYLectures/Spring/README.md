- ## 프로젝트 설명
    - JSP를 이해하고 SSR 형태의 웹을 구현한다.
    - MySql database와 연동하여 회원 관리를 진행한다.
    - MySql database와 연동하여 관광지 정보가 담긴 지도를 제작한다.
    - Session을 활용하여 페이지마다 로그인 유무를 확인하고 유저 정보를 가져온다.
